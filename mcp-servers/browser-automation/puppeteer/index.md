## Was ist Puppeteer?

Puppeteer ist eine Node.js-Bibliothek, die eine High-Level-API zur Steuerung von Chrome oder Chromium über das DevTools-Protokoll bietet. Es ermöglicht Headless (ohne UI) oder Headful (mit UI) Betrieb.
